# MazeGame
自学Java基础之后，边学边做通过使用Java AWT和Java Applet来构建迷宫游戏的用户界面，使用深度优先搜索算法、随机普利姆算法以及递归分割算法来生成迷宫，通过DFS和BFS可以帮助用户快速的找到迷宫出路。另外，为了增加游戏的趣味性，在游戏中添加了敌人定位追踪这样一个功能，当玩家处在敌人的监测范围之内，敌人会采用DFS来找到两点之间的路径，从而追踪玩家。
